# TakeFive Sound Pack

Drop these 7 files here (mp3, ~200-800ms each):

- `tap.mp3`     — soft brush-on-snare (navigation)
- `log.mp3`     — walking bass note (entry logged)
- `habit.mp3`   — ride cymbal ting (habit complete)
- `save.mp3`    — vinyl crackle + chord (journal saved)
- `open.mp3`    — piano key press (app open)
- `delete.mp3`  — bass slide (delete entry)
- `error.mp3`   — soft horn wah (gentle error)

## Free sources
- https://freesound.org (search: "jazz brush snare", "upright bass pluck", "ride cymbal")
- https://pixabay.com/sound-effects
- Splice free sample pack

The app works fine without these — sounds are skipped silently if missing.
